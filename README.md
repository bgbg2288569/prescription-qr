# 健保處方箋 OCR 轉 QR Code

凌曜藥局 - 處方箋智慧辨識系統

## 功能特色

✅ **拍照自動辨識** - 使用 Tesseract.js OCR 引擎辨識處方箋  
✅ **符合健保規範** - 完全遵照健保局 QR Code 格式標準  
✅ **繁體中文支援** - 支援繁體中文和英文辨識  
✅ **手機完全離線** - 網頁開啟後可離線使用  
✅ **即時生成 QR Code** - 立即產生可供電腦掃描的 QR Code  

## 使用方式

### 手機端操作：

1. 開啟網頁：在 iPhone Safari 瀏覽器輸入網址
2. 拍攝處方箋：點擊「啟動相機」或「選擇照片」
3. 等待 OCR 辨識：系統會自動辨識處方箋內容
4. 確認並修正：檢查辨識結果，必要時手動修正
5. 生成 QR Code：點擊「生成健保 QR Code」

### 電腦端掃描：

使用 WebCam 掃描 QR Code 即可取得處方資料

## 健保 QR Code 格式

符合衛福部健保署規範，包含以下欄位：

1. 醫事機構代號 (10碼)
2. 處方類型 (1=一般, 2=連續)
3. 案件分類 (2碼)
4. 病患姓名
5. 身分證字號 (10碼)
6. 出生日期 (民國年YYYMMDD)
7. 就醫科別 (2碼)
8. 就醫日期 (民國年YYYMMDD)
9. 健保卡序號 (4碼)
10. 給藥日份
11. 免部分負擔代碼
12. 國際疾病分類碼 (ICD)
13. 診治醫師代號
14. 備註
15-19. 藥品資料 (代號、用量、頻率、途徑、總量)

## 技術架構

- **前端框架**: Vanilla JavaScript (無需框架)
- **OCR 引擎**: Tesseract.js 4.x
- **QR Code**: QRCode.js
- **語言模型**: 繁體中文 + 英文

## 瀏覽器支援

- ✅ iOS Safari 12+
- ✅ Chrome 80+
- ✅ Edge 80+
- ✅ Firefox 75+

## 注意事項

⚠️ **OCR 辨識提示**：
- 確保光線充足
- 處方箋平整無皺摺
- 字跡清晰、對比度高
- 務必人工核對辨識結果

⚠️ **資料安全**：
- 所有處理均在瀏覽器本地執行
- 不會上傳任何資料到伺服器
- 不會儲存任何個人資料
